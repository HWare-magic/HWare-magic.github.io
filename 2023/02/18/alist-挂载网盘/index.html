alist 挂载网盘

发表于 2023-02-18 分类于 教程
挂载百度网盘到alist 以及挂载到本地电脑的教程

所需工具：1. alist 3.80版本 （最新的版本有会卡机的情况）

​ 2. raildrive 2022.6.92版本

分享连接：链接: https://pan.baidu.com/s/1ccfdIl8U3PZVYkv9yiOCWg?pwd=qgxb 提取码: qgxb

安装alist

1.解压后在alist文件夹处运行命令行窗口

2.在命令行窗口运行alist server 复制密码

3.在网页段打开链接 127.0.0.1:5244

4.点击页面下方登录按钮 用户名自己设置例如：admin 密码为第二步复制的密码

5.点击下方管理页面，再点击存储,在左侧点击添加

6.添加上百度网盘进行配置

7.挂载路径/任意名称， web代理需要打开 代理策略选择 使用代理地址，或者本地地址都可以。

8.下载接口这里选择官方的，非官方的失效了

9.其余的三个 刷新令牌 客户端ID 客户端密钥 需要打开网页版的百度网盘，然后

通过这个连接：百度网盘 | AList文档 (nn.ci)，点击刷新令牌来获取

这样填写好后，就可以点击保存，然后看到
